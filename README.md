## 1. 연관 아티스트 네트워크 그래프

배포링크: https://nextjs-d3-tailwind-i202ejby2-somedaycode.vercel.app/network

검색한 아티스트와 연관된 아티스트들로 네트워크 그래프를 그린다.

### 사용 기술

- NextJS
- React
- D3.js
- TailwindCSS

### 실행

```
# git clone https://github.com/somedaycode/nextjs-d3-tailwind

# yarn dev

# then go to localhost:3000/network
```
